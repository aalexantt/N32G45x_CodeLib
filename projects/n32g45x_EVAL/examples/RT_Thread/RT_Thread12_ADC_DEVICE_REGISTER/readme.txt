1、功能说明

    1、此例程展示在 RT_Thread 系统创建 ADC 设备

2、使用环境

    /* 硬件环境：工程对应的开发硬件平台 */
    开发板：N32G457QE_EVB V1.0

3、使用说明

    /* 描述相关模块配置方法；例如:时钟，I/O等 */
    SystemClock：144MHz
    GPIO:   LED:D6--PB10、D17--PB15
            KEY:S4--PC6(KEY1),S5--PA5(KEY2)
    ADC:
            PC0选择为模拟功能ADC1转换通道
            PC1选择为模拟功能ADC1转换通道
            
    日志：  TX--PA9
            

    /* 描述Demo的测试步骤和现象 */
    1.编译后下载程序复位运行
    2.本例程创建三个线程，开始线程、LED0 线程和 LED1 线程，开始线程用于创建其他线程（LED0和LED1），LED0 线程用于控制 D6 500ms闪烁，
      LED1 线程读取 KEY1 电平，高则 D17 亮，低则 D17 灭，KEY2 按下触发中断，并在回调函数中打印日志;
      在开始线程中测试ADC设备,在主线程中测量PC0和PC1引脚电压，通过串口打印结果。
